# PointOfSale Project - YM Clothing

A simple Point of Sale (POS) system built for YM Clothing store.  
This project helps manage products, sales, and inventory efficiently.

## Features
- Add, edit, and delete products
- Manage stock and inventory
- Record sales transactions
- Generate sales reports
- User authentication (admin and cashier roles)

## Tech Stack
- **Backend:** PHP, MySQL, Laravel
- **Frontend:** HTML, CSS, JavaScript
- **Other Tools:** XAMPP (local server)
